## â“ Troubleshooting

### Email Not Received?
1. **Check spam folder** (sometimes emails go there)
2. Click **"Resend Code"** button in app
3. Wait a few seconds
4. Check email again

### Code Expired?
- Verification codes expire after **10 minutes**
- Click **"Resend Code"** to get a new one
- **Start over**: New email, new code

### Wrong Password?
- Check **CAPS LOCK** is off
- Password is **case-sensitive**
- Minimum **6 characters** required
- Try again or use "Forgot Password" (coming soon)

### Receipt Upload Fails?
1. Make sure image is a **valid photo** (JPG, PNG, PDF)
2. File should be **under 50MB**
3. Receipt should show:
   - Product name
   - Purchase date
   - Price or warranty info
4. Try again or add manually

### Can't See Dashboard?
1. Check internet connection
2. Try refreshing page (F5 or Cmd+R)
3. Try logging out â†’ logging in again
4. Clear browser cookies and try again
